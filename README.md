# Task-3
🌡️ Temperature Converter
A simple and efficient temperature converter built with the HTML,CSS and JAVASCRIPT This project allows users to convert values between Celsius, Fahrenheit, and Kelvin using standard mathematical formulas.

🔧 How It Works
Input: Enter a temperature value and select the unit (C, F, or K).

Conversion Logic: The program applies the correct formula based on the chosen units:

Celsius → Fahrenheit: (C × 9/5) + 32

Fahrenheit → Celsius: (F - 32) × 5/9

Celsius → Kelvin: C + 273.15

Kelvin → Celsius: K - 273.15

Output: Instantly displays the converted temperature in the desired unit.
